### 改了什么行为

各服务具备 Batch 1 最小验收能力：

- **api**：`GET /health`；GraphQL `{ __typename }` / `health` / 域查询 stub；`/api/chat/*` 返回 501 占位
- **core**：HTTP `/health`；gRPC 端口监听（Ping 手写 stub，待 protoc 生成）
- **indexer**：`lingprism-indexer version`；`parse --language rust --file <path>` 输出符号 JSON
- **mcp**：`POST /mcp`（需 `MCP-Protocol-Version` + `X-API-Key`）；echo tool 可用
- **ai-worker**：`workers.ping` Celery task；LLM factory 占位（不调用真实 API）

### 为什么这样改

对齐计划 Batch 1：**五服务并行骨架、契约占位、可独立启动验证**；auth / 真实 LLM / SSE 业务留给 Batch 2–3。
